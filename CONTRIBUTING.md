# Contributing

If you'd like to make a contribution to this project, feel free to work on it straight away.

That being said, if yours is a large contribution, please discuss it with me beforehands. While it's unlikely that I'll reject your contribution, I wouldn't want your hard work to go to waste! 🙂

One way to contact me, preferably, is via creating an [issue](https://github.com/Aevarkan/dialogue-node-editor/issues) in this repository.

Feel free to reach out even if this repository seems inactive. If my account appears active, then I'll see any activity here.
